Nowadays, not only web developers and web designers, but also the general public are exposed to HTML and CSS through blogs and university liberal arts lectures. So, let's see what HTML and CSS are.

First, let's look at the definitions of HTML and CSS. HTML is an abbreviation of Hyper Text Markup Language, which is a markup language for making web documents. CSS is an abbreviation of Cascading Style Sheets, which is a style sheet that stores the overall style of web documents. For ease of explanation, let’s take an example with a building. HTML is the base of a building, and CSS is design elements such as a paint on the exterior of the building and wallpaper in the room. Now let's take a closer look at HTML and CSS.

HTML can be divided into four main components.

  1. Tag: It means a command consisting of a start tag and an end tag.
  2. Attribute:The start tag involves attributes and the attributes specify additional information
  3. Content: a content exposed on the website and directly displayed to users.
  4. Element: We call the whole from the start tag to the end tag a element.

We use the 4 components as shown in the picture below.

Tag names are always enclosed in < >, and end tags are appended with a slash. Not all tags have end tags. In the case of an element that does not contain content, there is no end tag. For example, the Img tag that loads an image has no end tag like the following sentence.

< img src=”file path”alt=”description of image”>

CSS consists of three factors as follows.

  1. Selector: To determine target element in HTML
  2. property: Kind of style for the target element
  3. value: Value of the property
